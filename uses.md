# @uses
## Использование

Показывает одновременно ссылки "с" и "на" ассоциированный структурный элемент.
Автоматически создает двухстороннюю ссылку между элементами, что очень удобно.
Отношения между текущим элементов и тем, на который ссылка.

@uses [_FQSEN_] [_описание_]

Пример:
```php
/**
* @uses MyClass::$items to retrieve the count from.
*
* @return integer Indicates the number of items.
*/
function count()
{
//<...>
}
```
